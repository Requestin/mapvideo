# Ручная проверка сценариев (QA чеклист)

Связанные документы: [task9.md](../task9.md) (деплой, curl), [SPEC.md](../SPEC.md).

---

## 1. Авторизация

- [ ] Логин с **верными** данными → открывается редактор (`/`), не остаётся на `/login`
- [ ] Логин с **неверным** паролем → **toast** с ошибкой, вход не выполняется
- [ ] Закрыть вкладку или браузер, снова открыть сайт → **сессия сохранена**, на `/login` не выкидывает
- [ ] **Выйти** из меню → редирект на `/login`, защищённые страницы недоступны
- [ ] **admin** открывает `/admin` → страница списка пользователей
- [ ] **Не admin** открывает `/admin` → редирект на `/` (или иная защита по коду)

---

## 2. Точки

- [ ] **+ Точка** → модалка; ввод **≥2** символов в поиске → **геосаджест**, выбор варианта → точка с координатами
- [ ] **Без подсказки**: ввести **название** в поле подписи, **Продолжить** → точка в **центре текущего вида** карты
- [ ] При сбое геокодера (если удаётся воспроизвести) → **toast** (предупреждение/ошибка)
- [ ] **3+ точки** → карта **автомасштабируется** под все точки
- [ ] **Перемещение точки** → линии маршрутов/зависимости обновляются
- [ ] **Перемещение подписи** независимо от маркера точки
- [ ] Все **4 анимации** точки (из SPEC) видны в превью
- [ ] **Удаление** точки с подтверждением → исчезновение, каскад маршрутов
- [ ] **Сброс местоположения** точки и/или подписи (кнопки в панели настроек)

---

## 3. Маршруты

- [ ] Кнопка **↗ Маршрут** при **≥2 точках**; пошаговое построение: первая точка → вторая точка или произвольная точка на карте
- [ ] Маршрут **между двумя** существующими точками отображается (линия, направление по SPEC)
- [ ] Маршрут **в произвольное место** (вторая нога — клик по карте не в точку)
- [ ] **Escape** — отмена режима построения, подсказка/курсор сбрасываются
- [ ] Транспорт **автомобиль** + маршрут **по дорогам** (OSRM): при успехе — дорога; при **недоступном OSRM** — прямая + **toast** о временной недоступности дороги
- [ ] **Самолёт** (и при необходимости другой вид без «по дороге») — **дуга** там, где по задумке не OSRM-линия
- [ ] Транспортная **иконка** движется и **разворачивается** по направлению
- [ ] **Удаление точки** на маршруте → маршрут исчезает или пересчитывается по правилам редактора

---

## 4. Настройки видео (⚙ Видео)

- [ ] Смена **темы** в модалке — карта в превью **сразу** отражает тему
- [ ] **Дыхание камеры** (ползунок) — эффект **только пока** открыта модалка
- [ ] Закрытие модалки (× / клик по оверлею / Esc / **Отмена**) — **без** сохранения тема/превью **откатываются** к снимку до открытия, твины останавливаются
- [ ] **Сохранить** — настройки фиксируются, согласованность с **тулбаром** (тема)
- [ ] **Длительность** вне **3–60** или невалидный ввод — **валидация** / кламп, как в UI

---

## 5. Рендер и «Моя история»

- [ ] **💾 Сохранить** — интерфейс **заблокирован**, виден **прогресс**
- [ ] При открытой вкладке прогресс **обновляется** (ожидаемо ~2 с)
- [ ] **Успех** — **скачивание** файла + **toast** об успехе
- [ ] **Ошибка** рендера или **429** — **toast** с причиной, блокировка снята
- [ ] **Закрыть** браузер на середине рендера, снова открыть `/` — **восстановление** активного job, прогресс отображается
- [ ] **Моя история** — готовая запись, **миниатюра**, **повторное скачивание**

- [ ] Создание пользователя в `/admin` с занятым именем — ошибка inline + toast (фикс 16.4)
- [ ] Удаление пользователя — подтверждение + при ошибке toast (без `window.alert`)
