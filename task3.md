## Роутинг

```typescript
// src/маршруты.tsx
<Routes>
  <Route path="/login" element={<СтраницаЛогина />} />
  <Route element={<ЗащищённыйМаршрут />}>
    <Route path="/" element={<СтраницаРедактора />} />
    <Route path="/admin" element={<АдминМаршрут><СтраницаАдмина /></АдминМаршрут>} />
  </Route>
</Routes>
```

```typescript
// ЗащищённыйМаршрут — проверяет токен, редиректит на /login
// АдминМаршрут — проверяет роль admin, редиректит на / если не admin
```

## Хук useAuth

```typescript
interface КонтекстАвторизации {
  пользователь: Пользователь | null;
  загрузка: boolean;
  войти: (логин: string, пароль: string) => Promise<void>;
  выйти: () => Promise<void>;
}

// При старте приложения:
// 1. Читаем токен из localStorage
// 2. Если есть — вызываем GET /api/auth/me для проверки
// 3. Если токен протух — чистим localStorage и редиректим на /login
```

## Меню пользователя

```typescript
// При клике на кнопку с именем — выпадает меню
// Клик вне меню — закрывается
// Анимация: плавное появление сверху вниз

function МенюПользователя() {
  return (
    <ВыпадающееМеню>
      <ПунктМеню onClick={открытьТехПоддержку}>
        Тех. поддержка
      </ПунктМеню>
      <ПунктМеню onClick={открытьИсторию}>
        Моя история
      </ПунктМеню>
      <РазделительМеню />
      <ПунктМеню onClick={выйти} опасный>
        Выйти
      </ПунктМеню>
    </ВыпадающееМеню>
  );
}
```

## Панель "Моя история"

- Открывается как drawer справа (слайд из-за края)

- Получает данные: GET /api/videos
