## Задачи

### 17.1 — Проверка пути перед удалением в cron-cleanup (п.8 ревью)

**Файл:** `backend/src/cron/render-cleanup.ts`

**Проблема:** `unlink` вызывается по путям из БД без проверки, что путь внутри `VIDEOS_ROOT`. Комментарий обещает проверку, но она не реализована.

**Решение:** Перед `unlink` проверять:

```typescript
function isSafeToDelete(filePath: string): boolean {
  const resolved = path.resolve(filePath);
  return resolved.startsWith(VIDEOS_ROOT + path.sep);
}
```

Если не прошло — логировать предупреждение, не удалять.

### 17.2 — Исправить MXF 50i / FFmpeg согласованность (п.10 ревью)

**Файл:** `backend/src/render/ffmpeg-args.ts`

**Проблема:** При fps=50 и MXF формате входные 50 кадров/сек могут не совпасть с выходными 25 fps + tinterlace.

**Решение:** Добавить `-framerate 50` в input args для MXF-50i сценария и убедиться, что `totalFrames = captureFps * duration` корректно. Добавить комментарий объясняющий логику.

### 17.4 — Исправить целостность labelId при удалении подписи (п.12 ревью)

**Файл:** `frontend/src/state/editor-state.tsx`

**Проблема:** При удалении подписи у точки остаётся `labelId` ссылающийся на несуществующий элемент.

**Решение:** При удалении label найти точку-владельца и обнулить `labelId`:

```typescript
if (target.kind === 'label') {
  return prev.map(e =>
    e.kind === 'point' && e.labelId === id ? { ...e, labelId: '' } : e
  ).filter(e => e.id !== id);
}
```

**Тест:** Тест в `editor-state.test.ts` — удалить подпись, проверить что `labelId` у точки пуст.
